### 📱 Screen 단위

- Login  : 로그인 화면
- AdminPageHome : 홈 화면
- AddPizza : 피자 추가 화면
- InfoList : 각종 정보(고객 정보, 직원 정보, 주문 정보)를 보여주는 리스트 화면 
- MenuAdminPage : 피자 메뉴들을 확인 할 수 있는 화면

<br>

## 👷 실행화면
 **- 기준 iPhone : 아이폰se2, 아이폰12mini, 아이폰12Pro**
